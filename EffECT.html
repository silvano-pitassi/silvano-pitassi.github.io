Efficient Eddy Current Testing (EffECT)

Project Description

EffECT is a two-year project funded by the Marie Skłodowska-Curie Actions. This project has two main goals:

  • To develop new mathematical and numerical tools for the analysis of Discrete De Rham (DDR) numerical schemes.
  • To apply these tools to real-world Eddy Current Testing configurations.

The central idea of the project is to integrate the novel mathematical capabilities of DDR methods, in particular their support for general polyhedral meshes and high-order polynomial approximations, into eddy current models.
